# Neel Sanghvi

Personal portfolio website showcasing my work as a Full Stack Developer. <br/>
Link - [neelsanghvi.dev](https://neelsanghvi.dev/)

## Technologies Used

- React + TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- shadcn/ui

## Getting Started

```sh
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
